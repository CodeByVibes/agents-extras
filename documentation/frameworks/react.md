# React Documentation Guidelines

1. **Component Documentation:** Document React components using JSDoc/TSDoc directly above the component declaration.
2. **Props Definition:** Document all props. If using TypeScript, document the Interface. If using JavaScript, use `PropTypes` and JSDoc.
3. **Storybook:** If the project uses Storybook, treat the stories as living documentation. Ensure every reusable component has a corresponding story.
4. **Hooks:** Document custom hooks explaining what state they manage and what effects they trigger.
